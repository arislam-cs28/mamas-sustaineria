# Mama's Sustaineria
Welcome to Mama's Sustaineria, an all new game featuring reusing, reducing, and best of all? Recycling! In our Godot game, we welcome the intersection of technology, arts, and sustainability. 
This game is created by Mallory, Arisha, Maggie, and Amy (the MAMA team)! Based off of the popular *Papa's Freezeria*.
The project is for our POE (Principles of Engineering) class.
Game made with Godot. Not suitable for mobile.
